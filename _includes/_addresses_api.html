Addresses

/addresses get

Get a list of addresses.

Query Parameters

  • USRN: (integer)

    Filter by USRN.

  • UDPN: (integer)

    Filter by UDPN.

  • Nearby filter: (string)

    Filter what is nearby based on a bounding circle.

  • Nearest filter: (string)

    Filter to the nearest point

HTTP status code 200

Body

Type: application/json

Schema:

{
  "$schema": "http://json-schema.org/draft-03/schema",
  "properties": {
    "uprn": {
      "required": true,
      "type": "integer"
    }
  },
  "required": true,
  "type": "object"
}

Example:

[
  {
    "@type": "LocalAuthorityAddress",
    "address": {
      "@type": "PostalAddress",
      "streetAddress": "5 Acacia Avenue",

      "latitude": "",

    }

  }
]
