## Development

### Build HTML from RAML

Install the raml2html tool:

    npm i -g raml2html

Run the tool:

    rake

### Running Jekyll locally

    git clone https://github.com/LocalGovDigital/Localo.git
    git co gh-pages
    bundle
    bundle exec jekyll serve -w
